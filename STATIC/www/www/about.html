About This Project

Webserv is a custom HTTP server implementation written in C++ 98.

Features Implemented:

  • HTTP/1.1 protocol support
  • Non-blocking I/O using poll() or select()
  • Multiple port listening
  • GET, POST, and DELETE methods
  • Static file serving
  • CGI execution (PHP, Python)
  • File upload handling
  • Custom error pages
  • HTTP redirections
  • Directory listing
  • Configuration file parsing

Technical Details:

Language: C++ 98

I/O Model: Non-blocking with poll()/select()

Compatibility: Works with standard web browsers

Testing Information

This server is designed to be tested with:

  • Web browsers (Chrome, Firefox, Safari)
  • curl command line tool
  • telnet for raw HTTP requests
  • Custom test scripts
