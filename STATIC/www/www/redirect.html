Redirect Information

This page should be configured in your server config to redirect to another location.

Example Configuration:

location /redirect.html {
    return 301 /index.html;
}

If you can read this, the redirect is not configured correctly, or you've disabled automatic redirects in your browser/client.

Test Cases:

  • 301 Moved Permanently - Permanent redirect
  • 302 Found - Temporary redirect
  • 307 Temporary Redirect - Temporary (maintains method)
  • 308 Permanent Redirect - Permanent (maintains method)
